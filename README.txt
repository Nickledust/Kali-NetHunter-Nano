# Kali-NetHunter-Nano README

Le fichier install-nethunter-termux est le même script necessaire à l'instalation de NetHunter sur Termux 
à l'exeption qu'il est fait pour telecharger la version nano (168MiB) au lieu de la version full (1.7Gib).

L'URL utilisé à la base lors de l'instalation de la version full est :
http://offs.ec/2MceZWr

L'URL à utilisé pour installer la version nano est :
https://cutt.ly/CBuA4Bb
C'est un lien raccourcci de la version raw du fichier install-nethunter-termux

La commande devrait ressembler à ca desormais :
wget -O install-nethunter-termux https://cutt.ly/CBuA4Bb
